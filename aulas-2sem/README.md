# Microservice and Web Engineering - 2o Semestre 2026

Aulas da disciplina **Microservice and Web Engineering** do curso de Sistemas de Informacao da FIAP, turma 3SIZ.

## Como usar

```bash
# Instalar dependencias
cd aulas-2sem
npm install

# Abrir as aulas
# Abra o arquivo aulas/index.html no navegador
# Ou use um servidor local:
npx serve .
```
